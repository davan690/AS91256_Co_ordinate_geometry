# Textbook Drafts

This folder contains scripts and outputs for generating textbook versions of the course materials.

## Files

- `generate_textbook.R` - R script that compiles all lessons into a single textbook document
- `generate_textbook.ps1` - PowerShell script that compiles all lessons (works without R)
- `textbook.qmd` - The compiled textbook (generated by either script)
- `textbook.html` - HTML version of the textbook (generated by Quarto, saved to docs/drafts/)
- `textbook.pdf` - PDF version of the textbook (generated by Quarto, requires LaTeX)

## Usage

### Generate the Textbook

**Option 1: PowerShell Script (Recommended)**

Run from the project root:

```powershell
powershell -ExecutionPolicy Bypass -File drafts\generate_textbook.ps1
```

**Option 2: R Script**

Run the R script from the project root:

```r
source("drafts/generate_textbook.R")
```

Or from RStudio, open `generate_textbook.R` and click "Source".

### Render the Textbook

After generating `textbook.qmd`, render it to your preferred format:

**HTML (recommended for web viewing):**
```bash
quarto render drafts/textbook.qmd --to html
```

**PDF (requires LaTeX installation):**
```bash
quarto render drafts/textbook.qmd --to pdf
```

**Both formats:**
```bash
quarto render drafts/textbook.qmd
```

### Using RStudio

1. Open `drafts/textbook.qmd` in RStudio
2. Click the "Render" button
3. Choose your preferred output format

## Customization

To customize the textbook appearance, edit the YAML header in `generate_textbook.R`:

- Change theme (HTML): `theme: cosmo` (options: default, cerulean, journal, flatly, etc.)
- Adjust margins (PDF): `geometry: margin=1in`
- Modify table of contents depth: `toc-depth: 3`
- Enable/disable section numbering: `number-sections: true`
